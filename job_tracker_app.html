Gmail Integration

Setup Instructions:
1. Go to Google Cloud Console
2. Create a new project or select existing one
3. Enable Gmail API (APIs & Services → Library → Search "Gmail API")
4. Create OAuth 2.0 credentials (APIs & Services → Credentials → Create Credentials)
5. Add your domain to authorized origins (e.g., http://localhost:8000)
6. Copy the Client ID and paste it above
